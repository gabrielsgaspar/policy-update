# Replication Checklist

## Before Circulating

- [ ] Every table/figure has a script.
- [ ] Every table/figure uses a documented dataset.
- [ ] Every clean/processed dataset traces to raw sources and code.
- [ ] Raw data is not manually edited.
- [ ] Source URLs, fetch timestamps, hashes, parser versions, and retrieval statuses are stored.
- [ ] Archive missingness and comment recovery are documented.
- [ ] Merge keys and merge losses are documented.
- [ ] Commenter identifiers are hashed in analysis data.
- [ ] Random seeds are fixed where relevant.
- [ ] LLM prompts, model versions, schemas, and validation status are logged.
- [ ] Coefficients in text match tables.
- [ ] Paper claims are supported by `docs/RESULTS_LOG.md` entries or generated outputs.
- [ ] LaTeX compiles from a clean checkout.
