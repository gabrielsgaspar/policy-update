\section{Data}\label{sec:data}

The planned data link four components: citizen input from comments on Beppe Grillo/M5S online spaces; intermediary party communication from Grillo/M5S posts, campaign messages, platform materials, and programs; formal political outputs such as parliamentary questions, bills, motions, speeches, and local or national programs; and external controls including media agenda, public search interest, election results, comparison-party outputs, and political events.

Raw data will be stored immutably, with provenance fields for source URL, fetch timestamp, archive timestamp where relevant, parser version, content hash, and retrieval status. Public comments will be treated as ethically sensitive, with commenter names hashed in analysis data and individual identities not inferred.

The first data task is to establish feasibility: whether historical comments can be recovered with usable timestamps or ordering, and whether archive missingness can be measured. See \texttt{DATA.md}, \texttt{docs/DATA\_SOURCES.md}, and \texttt{docs/DATA\_EXTRACTION\_PLAN.md}.
