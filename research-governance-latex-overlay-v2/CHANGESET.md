# Changeset

This overlay adds a PI-led research governance workflow for LaTeX academic papers.

## Added

- Cross-tool `AGENTS.md`.
- Claude Code instructions: `CLAUDE.md`.
- Claude Code agents: `.claude/agents/`.
- Claude Code commands: `.claude/commands/`.
- Cursor rules: `.cursor/rules/`.
- Codex skills: `.agents/skills/`.
- Durable research memory templates: `RESEARCH.md`, `docs/*.md`.
- Modular LaTeX paper scaffold: `paper/`.
- Lightweight checker: `scripts/research_doctor.py`.

## Important

For existing paper projects, merge rather than overwrite `paper/`, `AGENTS.md`, and any existing `docs/` files.
