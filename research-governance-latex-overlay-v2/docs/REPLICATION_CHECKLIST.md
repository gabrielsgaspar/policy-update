# Replication Checklist

## Before circulating

- [ ] Every table/figure has a script.
- [ ] Every table/figure uses a documented dataset.
- [ ] Outputs can be regenerated from clean scripts.
- [ ] The paper does not cite stale outputs.
- [ ] Coefficients in text match tables.
- [ ] Sample restrictions are documented.
- [ ] Merge losses are documented.
- [ ] Random seeds are fixed where relevant.
- [ ] LaTeX compiles from clean checkout.
