# Research Cadence

## Each session

1. Run `/research-brief` or ask Cursor/Codex for a research brief.
2. Choose one mode: build, critique, execute, write, review.
3. Execute one bounded task.
4. End with `/end-research-session`.

## Weekly

Run `/weekly-research-retro`.

Questions:

- What did we learn?
- What did we try?
- What worked?
- What failed?
- What is the current strongest paper version?
- What is the biggest unresolved threat?
- What should we stop doing?
- What is next week's plan?

## Monthly

Run a viability review:

- Is this still a paper?
- What is the publication path?
- What is the fatal unresolved issue?
- What would make us pivot, pause, or kill the project?
